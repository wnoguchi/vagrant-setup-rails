Rails Stack w/ Vagrant and Ansible
==========================================================

Requirements
--------------

- Vagrant 1.8.6 or later
- Ansible 2.1 or later
- VirtualBox 5.1.6
- UNIX compatible System(Mac, Linux, etc.)

Usage
-------

### First impression

Simply,

```
vagrant up
```

### Provision After bootup

If you edit Ansible Playbook, provision following:

```
vagrant provision
```
